# Минимум — план

Источник задач — этот файл; память чатов не используется.

## Выполнено

- Задача 12 «Движение» — тихие функциональные анимации на CSS: заполнение круга и сегодняшней ячейки полосы при тапе, opacity-fade при смене вкладок, сворачивание карточки разбора после решения (`motionLeave`), тихое «Сохранено» при сохранении формы. Глобальный `@media (prefers-reduced-motion: reduce)` отключает всё; конечные состояния достижимы мгновенно.
- Задача 12.1 «Движение: усиление» — амплитуда движения поднята до отчётливо заметной (диапазон 180–260 мс): scale-«просадка» круга и ячейки на тап, fade экрана с вертикальным подъёмом, отчётливое схлопывание карточки разбора по высоте.
- Задача 13 «Отделка» — визуальный слой: полная дата в тексте ожидания разбора (без «сент..»), единый вес стрелок ↑↓ и тумблеров с заметным гашением граничных, гармонизация типографики через токены (типо-шкала, радиусы, ритм секций), семантический тон `--warn` для «серия прервана» (приглушённый тёплый, не алармовый).

Исходный план проекта исчерпан. Дальнейшие визуальные правки — точечные по обратной связи владельца.
