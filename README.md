# Минимум

Персональный трекер ежедневного минимума — той неснижаемой планки, которая выполняется даже в худший день. Приложение видит только минимум (не «норму»): день фиксируется одной отметкой без оценок, картина складывается раз в неделю на тихом разборе. Никаких напоминаний, серий, очков и графиков — пропуск остаётся нейтральным событием, а повышение планки только предлагается и требует явного согласия.

Технически это PWA на ванильном JavaScript без сборки и зависимостей: пять файлов, localStorage как источник истины и тихое зеркало в IndexedDB на случай его исчезновения. После первого открытия работает полностью офлайн.

**Живая версия:** https://xbaox.github.io/minimum/

## Установка на iPhone

1. Открыть ссылку в Safari.
2. «Поделиться» → «На экран „Домой“».
3. Запускать с домашнего экрана: после первого открытия онлайн приложение работает без сети.

## Разработка

```
npm install
npm test
```

Файлы деплоя лежат в корне (`index.html`, `styles.css`, `app.js`, `sw.js`, `manifest.json`, иконки). Конституция проекта — `CLAUDE.md`.

## Данные

Пользовательских данных в репозитории нет: отметки живут только в localStorage и IndexedDB устройства. Перенос между устройствами — экспорт/импорт JSON внутри приложения.
